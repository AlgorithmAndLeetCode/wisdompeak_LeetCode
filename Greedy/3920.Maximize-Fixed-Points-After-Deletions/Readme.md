### 3920.Maximize-Fixed-Points-After-Deletions

本题的突破口在于，对于数组里的任意元素nums[i]=v，只要i>=v，那么该元素就有机会左移成为fixed point. 所以我们可以将它们先过滤挑出来。

在这些候选元素里面，根据题意，我们需要找这样一个从左往右的子序列：首先，它们的数值必须是递增的，这显然。其次，它们左移到达fixed point所需的位移，也必须是递增的（更确切地说是非递减的）。举个例子，加入有两个元素数值a<b，如果a需要左移3位到达fixed point，b只需要左移2位到达fixed point，那么当我们先调整a到达预期位置的时候，b必然就无法到达预期位置了。显然a与b不能同时选取。

由此，满足以上两个条件的最长子序列的长度就是我们需要的答案。这就是经典的二维偏序最长子序列。在一个数组里，每个数组元素有两个分量，分别是{nums[i], i-nums[i]}。求里面的最长子序列，保证第一个分量递增，第二个分量非递减。

具体算法就是将数组元素按照第一个分量顺序排序（如果该分量相同的情况下，按照第二个分量的逆序排序）。此时再在这个排序后的数组里，按照第二个分量，找最长非递减子序列即可。
